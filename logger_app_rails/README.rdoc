== Sample Rails 4.2.0 App using external logger messages
 
 run the follwing command to set up the app:

 bin/bundle install 
 bin/rake db:migrate
 bin/rake seed_books
 bin/rails server

 go to http://localhost:300
 Play around with different controller actions and check the console for custom messeges. 
 Check the logger.txt file in lib/assets/log/loggger.txt

 HttpLogger hooked up to create method in the books controller.
